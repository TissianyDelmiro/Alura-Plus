## 📝 Descrição do Projeto

Este projeto consiste na replicação da interface do serviço de streaming Alura Plus. Durante o desenvolvimento, foram aplicados conceitos fundamentais de HTML5 e CSS3, garantindo que o layout fosse fiel ao design proposto e adaptável a diferentes resoluções.

## 🚀 Funcionalidades

- **Layout Responsivo:** A página se ajusta perfeitamente a dispositivos móveis e desktops.
- **Seções Informativas:** Apresentação clara dos planos e benefícios do serviço.
- **Botões Interativos:** Estilização de chamadas para ação (CTAs) com efeitos de hover.
- **Animações Fluidas:** Aplicação de movimentos suaves para tornar a navegação mais dinâmica.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização avançada utilizando:
    * **Flexbox:** Para alinhamentos complexos e distribuição de espaço.
    * **Variáveis CSS:** Para facilitar a manutenção de cores e fontes.
    * **Media Queries:** Para garantir a responsividade.
    * **Keyframes:** Para a criação de efeitos de animação.
